# Glossary

- **UserScopedDb**: `{ db: D1Database, userId: string }` context enforcing user scoping.
- **Reading mode**: UI mode activated by `?doc=<id>`.
- **Ingestion log**: operational log of email/RSS ingestion outcomes.
- **Enrichment queue**: Cloudflare Queue for extraction/image-cache async jobs.
- **Single-user mode**: self-hosted auth mode with optional perimeter auth.
- **Multi-user mode**: SaaS auth mode using session-based sign-in.
