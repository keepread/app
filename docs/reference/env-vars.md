# Environment Variables (Reference)

Core runtime variables used across web/workers:

- `AUTH_MODE`
- `OWNER_EMAIL`
- `EMAIL_DOMAIN`
- `COLLAPSE_PLUS_ALIAS`
- `CF_ACCESS_TEAM_DOMAIN`
- `CF_ACCESS_AUD`
- `AUTH_SECRET`
- `BETTER_AUTH_URL`
- `RESEND_API_KEY`
- `RESEND_FROM_EMAIL`
- `BROWSER_RENDERING_ENABLED`
- `BROWSER_RENDERING_ACCOUNT_ID`
- `BROWSER_RENDERING_API_TOKEN`
- `BROWSER_RENDERING_TIMEOUT_MS`

See `docs/architecture/cloudflare-runtime.md` for where each is used.
